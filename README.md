# Players
TD 2 - Développement Front Avancé : AJAX & Programmation Orientée Objet
Ce projet a été réalisé dans le cadre du TD 2 (Partie 1 et 2) du cours de Développement Front. L'objectif est de manipuler des données externes (JSON/XML) et de les afficher dynamiquement en utilisant les classes JavaScript (ES6) et l'API Fetch.
